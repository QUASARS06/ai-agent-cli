# Quick Definition

A minimal Chrome Extension (Manifest V3) that provides word definitions upon selection or double-click.

## Installation
1. Go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `quick-def` folder

## Features
- Double-click or select any single word.
- Fetches definition from `dictionaryapi.dev`.
- Displays in a floating overlay in the top-right corner.
- Dismiss with `Esc` key or the `X` button.
